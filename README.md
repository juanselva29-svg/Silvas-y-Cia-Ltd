Silvas y Cia Ltd - Sitio Web Oficial
Este proyecto consiste en el desarrollo del sitio web oficial para Silvas y Cia Ltd, una empresa dedicada a servicios de logistica internacional y gestion de documentacion.

Tecnologias utilizadas
El sitio ha sido desarrollado utilizando las siguientes tecnologias:
HTML5: Estructura semantica del contenido.
CSS3: Diseno responsivo y estilizacion visual.
JavaScript: Funcionalidades interactivas para el usuario.

Caracteristicas principales
-Diseno Responsivo: Adaptable a dispositivos moviles, tablets y computadoras de escritorio.
-Interfaz Intuitiva: Enfocada en facilitar al cliente la consulta de informacion sobre servicios logísticos.

Despliegue
El sitio se encuentra alojado en Hostinger.
