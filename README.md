# SILAGAD - Agencia Aduanera Silva & Cía. Ltda.

Bienvenido al repositorio oficial del sitio web de **SILAGAD**, una agencia aduanera dedicada a ofrecer soluciones integrales en logística, importación, exportación, asesoría fiscal y despacho aduanal.

## 🌐 Descripción del proyecto

Este proyecto corresponde a una **página web B2B** diseñada para presentar los servicios de la empresa, mostrar información sobre su trayectoria, facilitar la comunicación con clientes y ofrecer acceso rápido a canales de contacto.

## 📄 Contenido de la web

El sitio incluye las siguientes secciones:

- **Inicio**: presentación institucional, slider principal, mensajes clave y acceso rápido a servicios.
- **Nosotros**: historia, misión, visión, valores, experiencia y métricas destacadas.
- **Servicios**: información sobre importaciones, exportaciones, asesoría fiscal y aforos/liquidaciones.
- **FAQs**: respuestas a preguntas frecuentes sobre procesos aduaneros y documentación.
- **Contacto**: formulario de contacto, información de ubicación, teléfonos, correo y mapa.
- **Política de privacidad**: detalles sobre el manejo de datos personales.
- **Términos y condiciones**: normativa y uso del sitio.

## ✨ Funcionalidades principales

- Diseño responsivo para dispositivos móviles y escritorio.
- Navegación intuitiva entre páginas del sitio.
- Formularios integrados para contacto y suscripción.
- Botones de contacto directos con WhatsApp.
- Secciones de contenido optimizadas para presentación institucional.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- jQuery
- Font Awesome
- Owl Carousel
- WOW.js
